Oh hey, a new post. That can only mean one thing: time to rebuild the blog with a new framework.

This round it's [Astro](https://astro.build). Elevator pitch: it's a content-first web framework that ships mostly plain HTML by default, so pages stay fast and the JavaScript only shows up when you actually need it.

Very into it so far.

Mostly motivated by GitHub's security nag emails about my ancient Gatsby v1 install — rather than slog through the upgrades, I figured it was a good excuse to try something new. I used to be pretty excited about GraphQL, but after spending more time with it over the years, it feels like overkill for a blog. Astro seems to strike a much nicer balance than Gatsby did for my use case.

See you in another —checks notes— 5–10 years(!) for the next framework and blog post.
